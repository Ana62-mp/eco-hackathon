# eco-hackathon

# Descripción de proyecto
- EcoCalcWeb (ECW) es una aplciación web que ayuda a las personas a conocer su impacto ambiental diario. El usuario responde preguntas sobre transporte, consumo de energía, agua y residuos. Al final Python procesará esas respuestas y cálculara un nivel de impacto ambiental: bajo, medio o alto, además mostrará consejos prácticos para mejorar sus hábitos.
- El objetivo del proyecto es crear conciencia sobre el cambio climático y motivar pequeñas acciones cotidianas que contribuyan al cuidado del planeta.

## Este es el tipo de mi proyecto:

> EcoCalcWeb es un sitio web interactivo desarrollado con Flask. El proyecto funciona como una eco-calculadora que permite al usuario responder preguntas sobre sus hábitos diarios y recibir un resultado sobre su impacto ambiental.
